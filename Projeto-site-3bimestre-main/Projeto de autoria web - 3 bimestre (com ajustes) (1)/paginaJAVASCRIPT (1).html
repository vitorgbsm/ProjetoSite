POR FIM, IREMOS ESTUDAR JAVASCRIPT:

JavaScript (JS) é uma linguagem de programação popular usada principalmente para adicionar
interatividade e dinamismo a páginas da web, tornando-as mais do que apenas documentos estáticos.


O JavaScript é a linguagem que torna a web interativa, sendo essencial para o desenvolvimento Front-end e, graças ao Node.js, também Back-end.
O que é o NODE.JS?
• É um ambiente de runtime JavaScript de código aberto e multiplataforma
que permite executar código JavaScript fora do navegador, como em servidores.





1-Fundamentos da linguagem

Variáveis e Tipos:
-> Declaração: Use const (para valores que não mudam) e let (para variáveis que serão reatribuídas). Evite var.
-> Tipos Primitivos: String, Number, Boolean, null, undefined, Symbol.
-> Tipos Complexos: Object (que inclui Array e Function).
-> Coerção de Tipos: JS é fracamente tipada. Prefira sempre os operadores de comparação estrita (=== e !==) para evitar conversões automáticas inesperadas (que ocorrem com ==).
Funções:
-> São blocos de código reutilizáveis.
-> Arrow Functions (ES6): Sintaxe concisa (const func = () => {}) muito popular, que possui um comportamento lexical diferente para o contexto (this).

2-Estruturas de Dados e Manipulação

Arrays:
-> Coleções ordenadas.
-> Métodos Essenciais:
• Imutáveis (Funcionais): map() (transforma itens), filter() (seleciona itens), reduce() (combina o array em um único valor).
• Mutáveis: push(), pop(), splice().
Objetivos:
-> Coleções de pares chave-valor, usadas para modelar entidades.
-> ES6+: Desestruturação (Destructuring) (const { prop } = obj;) e Spread Operator (...) para clonar ou combinar objetos e arrays de forma limpa.

3-Interação com a página (DOM)

O DOM (Document Object Model) é a representação em árvore da página HTML. O JavaScript o utiliza para manipular a interface.
APRENDA MAIS SOBRE DOM!
-> Seleção: document.querySelector() e document.querySelectorAll() são os métodos modernos e mais flexíveis, usando seletores CSS.
-> Eventos: A forma de reagir a ações do usuário. Usa-se elemento.addEventListener('click', minhaFuncao).
-> Manipulação: Alterar o conteúdo (.textContent, .innerHTML), atributos (.setAttribute) e estilos (.style.propriedadeCSS).

4-Assincronicidade(O mais crucial)

Como o JS é single-threaded (executa uma coisa por vez), operações demoradas (como requisições de rede) são executadas de forma assíncrona para evitar travamento da página.
-> Promises: Objetos que representam o resultado futuro de uma operação (sucesso ou falha). Usam .then() e .catch().
-> Async/Await: Sintaxe moderna (açúcar sintático) sobre Promises. Permite escrever código assíncrono que parece síncrono, melhorando a legibilidade.
• async function buscaDados() { ... }
• const dados = await fetch('url');
-> Fetch API: O padrão moderno para fazer requisições HTTP e se comunicar com APIs externas.





5-Ecossistemas e padrões modernos

-> Módulos (ES Modules): O sistema import / export para estruturar o código em arquivos separados, facilitando a reutilização e manutenção.
-> Classes: Sintaxe baseada em protótipos (class MinhaClasse {}) para implementar a Programação Orientada a Objetos (POO).
-> Programação Funcional: Adoção crescente de funções puras e métodos imutáveis (como map, filter, reduce) para criar código mais previsível e fácil de testar.
