O QUE SÃO AS TAGS NO HTML?

Tags são rótulos usados para informar ao navegador como deve ser apresentado o website; • Todas as tags têm o mesmo formato: começam com um sinal de menor "<" e acabam com um sinal de maior ">"; • A forma corrta de escrever as tags é em letras minúsculas;



HTML



• Estrutura básica: é o esqueleto de um documento html, que contém as tags essenciais para definir o tipo do documento, incluindo as tags ("HTML"(Indica que estamos criando uma página HTML)) ("HEAD" (cabeça), essa tag fornece informações sobre o documento.) ("BODY" (corpo),essa tag abriga o conteúdo do documento.) e outras metatags,em cada uma delas deve estar entre( < > ).






• Para dar um título ao documento, você deverá usar a seção"head". - A tag para acresentar um título é: ("title") - O título deve aparecer no topo da barra do navegador.








• OUTRAS TAGS DE TEXTO: Tag : Informa ao navegador que todo o texto colocado entre e deve ser mostrado em itálico.
⇨ Tag ("br"): Serve para criar uma quebra de linha.
⇨ Tag ("hr"): Serve para definir uma linha horizontal.
⇨ Tag ("p"): Cria um novo parágrafo
LEMBRANDO: em cada uma delas deve estar entre( < > ).








• Tags semânticas: são tags que dão significado ao conteúdo da página, facilitando a compreensão dos usuários e dos mecanismos de busca. Por exemplo, usar a tag ("HEAD") para o cabeçalho do site, a tag ("NAV") para a navegação, a tag ("SECTION") para as seções principais, a tag ("A") pode sser usada para colocar um link na página e também fazer uma imagem,palavra e símbolos virar link,redirecionando para o lugar que o programador deseje. LEMBRANDO: em cada uma delas deve estar entre( < > ).







• Formulários: são elementos que permitem coletar dados dos usuários através de campos de entrada (input), botões (button), caixas de seleção (select), etc. É preciso definir o método (get ou post) e a ação (url) do formulário, além de usar tags como ("label"), ("fieldset") e ("legend") para organizar os campos. LEMBRANDO: em cada uma delas deve estar entre( < > ).










• Multimídia: são elementos que permitem inserir conteúdos multimídia na página, como imagens ("IMG"), áudios ("AUDO"), vídeos ("VIDEO") eiframes (iframes). É preciso definir os atributos como src (fonte), alt (texto alternativo), width (largura), height (altura), etc. LEMBRANDO: em cada uma delas deve estar entre( < > ).

O que é atributo? – O atributo serve para permitir que algumas tags possam ser mais específicas.
– Nas tags podem ser acresentadas informações adicionais.

RGB é a abreviatura do sistema de cores aditivas formado por Vermelho (Red), Verde (Green) e Azul (Blue). -Uma cor no modelo de cores RGB pode ser descrita pela indicação da quantidade de vermelho, verde e azul que contém.



As principais formas de representação são:
⇨ Decimal (0 a 255): Exemplo → (255, 0, 0) = vermelho;
⇨ Percentual (0% a 100%): Exemplo → (100%, 0%, 0%) = vermelho;
⇨ Hexadecimal (00 a FF): Exemplo → FF0000 = vermelho;
⇨ Nos programas de edição de imagem, os valores são habitualmente representados por meio de notação hexadecimal, indo de 00 (maisescuro)
até FF (mais claro) para o valor de cada uma das cores.


• Boas Práticas de Acessibilidade: são recomendações para tornar o código html mais limpo, organizado e acessível para todos os usuários, incluindo aqueles com deficiências ou limitações. Algumas dicas são: usar indentação, comentários, nomes claros e descritivos para os elementos e atributos, evitar tags obsoletas ou desnecessárias, usar atributos como title, alt e aria para fornecer informações adicionais, etc.LEMBRANDO: em cada uma delas deve estar entre( < > ).
